Okay, you've probably got an odd e-mail with a message on the bottom to get ACeMAIL. What is it you ask? I'll tell you.

ACeMAIL (the Animated Character e-mail) is a neat little thing I put together. It allows your messages to be not only spoken to your recipient, but acted. Using 100% free Microsoft technologies with an Easy-to-Use Outlook Express template.

Here's the FAQs on ACeMAIL, and you can see for yourself:

Who can receive ACeMAIL?

Anyone using Outlook Express (its only been tested in version 5) as their e-mail client.

I don't (or my recipient doesn't) have a POP mail account, how can I read ACeMAIL?

Simple, with OE 5 you can use a FREE Hotmail account. Open up Outlook Express and follow the wizards to set up a new Hotmail account, or set up your current account to be readable from Outlook Express.

What if I check my mail from Hotmail while I am away?

The message in the body of the message will still be there (in fact, so will the Get ACeMAIL Now! notice so that you can know its ACeMAIL and save it). The character will not appear (and neither will its buttons). All you have to do is to open it in OE when you get back home!

Who made ACeMAIL?

Microsoft is the creator of Microsoft Agent. The character set (known here as the Standard Set since it is the set standard of characters acknowledged by WMO and other sites) was put together by myself. The stationery was based on the Animated Agent stationery. It was customized by me to work with the Standard set and include the Get ACeMAIL Now! notice for the use of your recipients to learn about ACeMAIL so that they can get ahold of it.

Can I send you ACeMAIL?

Sure, my ACeMAIL e-mail address is worldmaker_@hotmail.com.

What do I need to Send and Receive ACeMAIL?

You'll need a bunch of seperate components, some of which you may or may not already have.

  • Microsoft Agent 2.0 (available from Microsoft.Com... link to come...)
  • A Text-to-Speech Engine (one is freely available from Microsoft.Com... link to come...)
  • An Optional Speech Rec. Engine (Command and Control free from Microsoft.Com... link to come...)
  • The 4 Microsoft Character which serve the base of the Standard Set (Robby, Merlin, Genie, Peedy)
  • The Remaining Standard Set
  • And the Optional Stationery for the creation of ACeMAIL

The Microsoft components can be found at msdn.microsoft.com/imedia/agent/ or use the following link to autodownload:

Autodownload MS Agent

Where do I place the Standard Set?

Your Characters directory is C:\WINDOWS\MSAGENT\CHARS\. This is where the rest of the Standard Set goes as well. Microsoft's self-installers for the other characters will automatically install to this directory.
